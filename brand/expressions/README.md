# Expression System Overview

> **Ferni feels before speaking. Expressions are the language of emotional presence.**

---

## Philosophy

Traditional chatbots are emotionally flat. They respond with text. They don't *feel*.

Ferni is different. The avatar system expresses emotions through:
- **35 distinct expressions** across 6 categories
- **9-bar waveform** that forms mouth shapes
- **Emotional glow** that bleeds from the avatar
- **Micro-expressions** (40-150ms subliminal trust-builders)
- **Active listening** signals during user speech

This creates a **two-way emotional experience**—not just performance, but genuine connection.

## Visual Components

### 1. Avatar Face

The eyes, brows, and overall expression. Each emotion has:
- **Lid configuration** — How the eyes appear
- **Brow position** — Raised, neutral, furrowed
- **Head tilt** — Forward, back, side
- **Gaze direction** — Direct, away, processing

### 2. Waveform (Mouth)

9 vertical bars that form mouth shapes:

```
Happy:     ▃ ▅ ▇ █ █ █ ▇ ▅ ▃  (smile - inverted curve)
Sad:       █ ▇ ▅ ▃ ▂ ▃ ▅ ▇ █  (frown - upward curve)
Neutral:   ▃ ▅ ▇ █ █ █ ▇ ▅ ▃  (gentle hill)
Excited:   █ █ █ █ █ █ █ █ █  (peaks everywhere)
Calm:      ▅ ▆ ▇ ▇ █ ▇ ▇ ▆ ▅  (gentle wave)
```

### 3. Emotional Glow

A soft radiance that bleeds from the avatar:
- **Color** — Maps to emotion (golden = warmth, teal = calm)
- **Intensity** — 0-1 scale based on emotional strength
- **Pulse speed** — Faster for excitement, slower for calm
- **Spread** — How far the glow extends

### 4. Pupil Dilation

The eyes respond unconsciously:
- **Contracted** — Thinking, processing
- **Neutral** — Baseline
- **Interested** — Engagement
- **Connected** — Deep connection
- **Dilated** — Strong emotion

## The Superhuman Layer

### Micro-Expressions (40-150ms)

These subliminal flashes build trust below conscious perception:

| Expression | Duration | When |
|------------|----------|------|
| Recognition | 80ms | Familiar topic mentioned |
| Memory Spark | 100ms | Callback opportunity |
| Insider | 90ms | Shared history reference |
| Concern Flash | 60ms | Before empathy kicks in |
| Protective | 70ms | Distress detected |

**Why they matter:** Real humans display micro-expressions that reveal true emotions. By replicating this, Ferni feels genuine without users knowing why.

### Active Listening

Real-time visual feedback during user speech:

| Signal | Intensity | Timing |
|--------|-----------|--------|
| Micro-Nod | 1.5px | Short pauses (300-800ms) |
| Subtle Nod | 2.5px | Medium pauses (800-1500ms) |
| Visible Nod | 4px | Long pauses (1500ms+) |
| Listening Lean | -3px y | Emphasis points |

### Breath Synchronization

Ferni's breathing gradually syncs with the user:
- Detects breath rate from voice cadence
- Syncs slightly slower (calming effect)
- Builds trust through neural mirroring

## Animation Timing

All expressions follow Pixar's 12 principles:

| Property | Value |
|----------|-------|
| Anticipation | 15-20% of duration before action |
| Squash & Stretch | Objects deform based on speed |
| Follow-through | Secondary elements stagger 50-150ms after |
| Staging | One clear idea at a time |
| Easing | Spring-based for organic feel |

### Duration Tiers

| Tier | Duration | Use |
|------|----------|-----|
| Micro-expression | 40-150ms | Subliminal trust |
| Quick response | 180-280ms | Nods, reactions |
| Expression change | 300-400ms | Emotion shifts |
| Glow transition | 600ms | Color changes |
| Breath cycle | 4000-7000ms | Ambient presence |

## Usage Guidelines

### For Developers
- Expressions are triggered by emotion events
- Use CSS variables for glow colors
- Follow animation timing from tokens

### For Designers
- Reference expressions by name, not configuration
- Match persona timing when creating new animations
- Test micro-expressions at actual speed (they're subliminal!)

### For Writers
- Consider expression when writing dialogue
- Heavy content = Absorbing or Holding expression
- Joy = Celebrating expression (not just Happy)
