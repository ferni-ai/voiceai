# Discord Server Structure

> **The Ferni Community Hub** - Where growth happens together.

---

## Server Philosophy

This isn't a support forum. It's a **community of practice**—people committed to growth, sharing their journey, and supporting each other. Ferni is the thread that connects them, but the humans are the heart.

**Core Values:**
1. **Be human** - Authenticity over performance
2. **Respect privacy** - What's shared here stays here
3. **Support growth** - Celebrate wins, hold space for struggles
4. **No judgment** - Every journey is valid
5. **Constructive feedback** - Help Ferni get better

## Channel Structure

### 🏠 WELCOME

#### #welcome
**Purpose:** First impression and onboarding
**Permissions:** Read-only for members, write for admins
**Content:**
```markdown
# Welcome to the Ferni Community 🌿

You're here because you believe in growth.
So do we.

**What this place is:**
- A community of people working on themselves
- A space to share reflections, wins, and struggles
- Direct access to the team building Ferni

**What this place isn't:**
- A support ticket system (use in-app support)
- A place to complain without constructive intent
- Somewhere to promote other products

**Get started:**
1. Read #rules
2. Introduce yourself in #introductions
3. Find your people in the persona channels

See you around. 💚
```

#### #rules
**Purpose:** Community guidelines
**Permissions:** Read-only
**Content:**
```markdown
# Community Guidelines

1. **Be kind.** We're all growing. No judgment.
2. **Respect privacy.** Don't share others' stories outside this server.
3. **Stay constructive.** Feedback welcome, complaints without solutions aren't.
4. **No self-promotion.** This isn't a marketing channel.
5. **Support each other.** If someone shares something vulnerable, respond with care.
6. **Use content warnings.** For heavy topics, use spoiler tags with CW.
7. **No politics.** There are other places for that.
8. **Report concerns.** DM a mod if something feels off.

Breaking these guidelines = warning → timeout → ban.
We want this to be a safe space. Help us keep it that way.
```

## Moderation Guidelines

### Response Times
- **Reported content:** < 1 hour
- **New member approval:** < 24 hours
- **Feature request acknowledgment:** < 48 hours

### Escalation Path
1. Warning DM from bot
2. Warning DM from moderator
3. 24-hour timeout
4. 7-day timeout
5. Permanent ban

### Content Requiring Immediate Action
- Spam / self-promotion
- Harassment / bullying
- Sharing private information
- NSFW content
- Crisis language (escalate to trained responder)

### Crisis Protocol
If someone expresses:
- Suicidal ideation
- Self-harm
- Immediate danger

**Action:**
1. Respond with empathy
2. Share crisis resources (pinned in #struggles)
3. DM Team immediately
4. Do not leave conversation abruptly
