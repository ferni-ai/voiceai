## Research Program

### Phase 1: Component Isolation

Test each Ferni EQ component individually.

#### Study 1.1: Micro-Expressions

**Question:** Do subliminal micro-expressions (40-150ms) build trust?

**Method:**
- A/B test with 1000 users
- Group A: Full micro-expression system
- Group B: Static expressions only
- Duration: 2 weeks

**Measures:**
- Trust scale (5-point Likert, post-session)
- Net Promoter Score
- Session count over 2 weeks
- Qualitative interviews (n=20)

**Protocol:**
```
1. User completes onboarding
2. Randomly assigned to Group A or B
3. Uses Ferni for 2 weeks
4. Post-study survey
5. Optional interview
```

#### Study 1.2: Active Listening

**Question:** Does real-time listening feedback improve perceived engagement?

**Method:**
- A/B test with 800 users
- Group A: Micro-nods, forward lean, gaze tracking
- Group B: Static listening pose
- Duration: 2 weeks

**Measures:**
- "Ferni was really listening" (5-point scale)
- Average session duration
- User speech time per session
- Topic depth (NLP analysis)

#### Study 1.3: Breath Synchronization

**Question:** Does breath sync create a sense of connection?

**Method:**
- A/B test with 600 users
- Group A: Breath sync enabled (gradual mirroring)
- Group B: Random breathing rhythm
- Duration: 1 week

**Measures:**
- Connection rating (5-point scale)
- Comfort rating (5-point scale)
- Biometric correlation (if feasible: HRV alignment)
- Qualitative: "Felt like we were in sync"

#### Study 1.4: Concern Detection

**Question:** Does proactive concern acknowledgment feel supportive vs. intrusive?

**Method:**
- Within-subjects design, 400 users
- Session 1: Concern detection ON
- Session 2: Concern detection OFF
- (Counterbalanced)

**Measures:**
- "Ferni noticed I was struggling" (5-point scale)
- "Felt supported" (5-point scale)
- "Felt surveilled" (inverse measure)
- Distress disclosure rate

#### Study 1.5: Anticipatory Emotion

**Question:** Does showing emotion before user finishes create connection or feel presumptuous?

**Method:**
- A/B test with 600 users
- Group A: Anticipation enabled
- Group B: React only after utterance complete
- Duration: 1 week

**Measures:**
- "Ferni understood before I finished" (5-point scale)
- "Ferni jumped to conclusions" (inverse)
- Conversation flow rating
- Interruption perception

### Phase 2: System Integration

Test combined Ferni EQ system vs. baseline.

#### Study 2.1: Full EQ vs. Static

**Question:** Does the complete Ferni EQ system outperform a static avatar?

**Method:**
- A/B test with 2000 users
- Group A: Full Ferni EQ (all 5 components)
- Group B: Static avatar (breathes only)
- Duration: 4 weeks

**Measures:**
- Overall connection score
- Trust score
- Session frequency
- Session duration
- 30-day retention
- NPS

**Statistical Power:**
- Detecting 15% difference
- 95% confidence
- Power: 0.8
- Required n: ~800 per group

#### Study 2.2: Circadian Integration

**Question:** Does time-aware theming improve user experience?

**Method:**
- A/B test with 1000 users
- Group A: Circadian theming (adapts to time of day)
- Group B: Static theme
- Duration: 3 weeks

**Measures:**
- Late-night session comfort
- Morning energy rating
- Theme preference survey
- Sleep quality impact (self-report)

#### Study 2.3: Relationship Depth

**Question:** Does progressive UI complexity improve retention?

**Method:**
- Longitudinal study, 500 users
- Track UI adaptation over 60 days
- Correlation analysis: complexity vs. engagement

**Measures:**
- Feature discovery rate
- Time to Stage 5
- Retention by stage
- Satisfaction by stage

---

### Phase 3: Competitive Benchmarking

Compare Ferni to alternative approaches.

#### Study 3.1: Ferni vs. Text-Only

**Question:** How much does visual presence add?

**Method:**
- A/B test with 500 users
- Group A: Full Ferni experience
- Group B: Text-only chat interface
- Duration: 2 weeks

**Measures:**
- Perceived connection
- Emotional disclosure rate
- Session depth
- Return rate

#### Study 3.2: Ferni vs. Standard Avatar

**Question:** Does Ferni's character design outperform generic alternatives?

**Method:**
- A/B test with 500 users
- Group A: Ferni avatar
- Group B: Generic animated circle
- Both with same animation system
- Duration: 2 weeks

**Measures:**
- Character appeal
- Anthropomorphization
- Trust
- Preference

### Survey Instruments

#### Trust Scale (adapted from Mayer et al.)

```
Please rate your agreement with each statement (1-5):

1. I believe Ferni has good intentions toward me.
2. I can rely on Ferni to keep things I share private.
3. Ferni seems to genuinely care about my well-being.
4. I feel comfortable being vulnerable with Ferni.
5. I trust Ferni's guidance and suggestions.
```

#### Connection Scale (developed for Ferni)

```
Please rate your agreement with each statement (1-5):

1. Talking to Ferni feels like talking to a real friend.
2. Ferni seems to understand me on a deep level.
3. I feel emotionally connected to Ferni.
4. Ferni makes me feel seen and heard.
5. Our conversations feel meaningful, not transactional.
```

#### Ferni EQ Perception Scale

```
Please rate your agreement with each statement (1-5):

1. Ferni noticed my emotional state without me saying it.
2. Ferni's responses matched how I was feeling.
3. Ferni seemed genuinely present during our conversation.
4. Ferni's expressions felt authentic, not robotic.
5. I sometimes forgot I was talking to an AI.
```

## Analysis Plan

### Quantitative Analysis

1. **Primary comparisons**: Independent t-tests for A/B tests
2. **Multi-factor**: ANOVA for component interaction effects
3. **Longitudinal**: Mixed-effects models for retention data
4. **Correlation**: Relationship between EQ perception and retention

### Qualitative Analysis

1. **Thematic analysis** of interview transcripts
2. **Sentiment analysis** of open-ended survey responses
3. **Behavioral coding** of session recordings (opt-in)

### Statistical Standards

- **Significance level**: α = 0.05
- **Multiple comparison correction**: Bonferroni for primary outcomes
- **Effect size reporting**: Cohen's d for all comparisons
- **Confidence intervals**: 95% for all estimates

## Ethical Considerations

### Informed Consent

All participants will:
1. Be informed they are in a research study
2. Know they may be in a control group
3. Have right to withdraw at any time
4. Understand data collection practices

### Privacy Protection

1. All data anonymized before analysis
2. Conversation content never stored with identifiers
3. Opt-in only for session recordings
4. Aggregated reporting only

### Vulnerable Populations

1. Screen for acute mental health crisis
2. Provide crisis resources in all conditions
3. Extra review for distress-related studies
4. Debrief for participants who disclosed

### Deception Policy

No deception will be used. Participants will know:
- They are testing Ferni's features
- They may see different versions
- Specific hypothesis is not revealed (to prevent bias)
