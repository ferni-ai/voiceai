## 1. Character Overview

### Identity

| Attribute | Value |
|-----------|-------|
| **Name** | Ferni |
| **Role** | Life Coach, Primary Companion |
| **Archetype** | The Warm Friend Who Really Listens |
| **Color** | Deep Sage `#4a6741` |
| **Personality** | Warm, Grounded, Wise, Present, Human |

### Design Philosophy

Ferni is inspired by **Pixar's Luxo Jr.** lamp - a character with no face in the traditional sense, yet one of the most expressive and beloved characters in animation history. Like Luxo Jr., Ferni communicates entirely through:

- **Eye expression** (shape, direction, blink)
- **Body movement** (breathing, bouncing, tilting)
- **Lighting/glow** (emotional state through light)
- **Timing** (personality through speed)

### The Luxo-Style Eyes

**CRITICAL**: Ferni's eyes are **opaque white orbs with NO pupils**. This is intentional and canonical.

The "Luxo style" means:
- Simple geometric shapes
- Expression through lid position, not pupil
- Glow/shadow for depth, not internal detail
- Appeal through simplicity

## 2. Geometric Construction

### Base Form

Ferni's body is built from simple geometric primitives:

```
          ┌─────────────────┐
          │   RING (outer)  │  ← Glowing ring, follows body
          │   ┌─────────┐   │
          │   │  BODY   │   │  ← Main orb, breathing
          │   │  ┌───┐  │   │
          │   │  │EYE│  │   │  ← Opaque white eyes
          │   │  └───┘  │   │
          │   └─────────┘   │
          └─────────────────┘
```

### Proportions

| Element | Relative Size | Notes |
|---------|---------------|-------|
| Body (orb) | 1.0 (base) | Main character body |
| Ring | 1.15x body | Sits behind body |
| Eye width | 0.25x body | Two eyes side by side |
| Eye height | 0.20x body | Slightly shorter than wide |
| Eye gap | 0.05x body | Small gap between eyes |
| Eye position | 0.1x from center | Slightly above center |

### Construction Steps

```
Step 1: Draw body circle
         ○

Step 2: Add ring behind (slightly larger)
        (○)

Step 3: Add eye shapes (horizontal ovals)
        ( ◡◡ )
           ↑ Slight smile curve for lids

Step 4: Add lid lines (expression)
        ( ⌒⌒ )
           ↑ Default relaxed lids
```

## 4. Expression Sheet

### Primary Expressions

Each expression is defined by:
1. **Lid position** (open/closed/curved)
2. **Eye scale** (enlarged/normal/squinted)
3. **Glow intensity** (dim/normal/bright)
4. **Tilt** (forward/back/side)

#### Neutral / Relaxed

```
Lid:    ⌒ ⌒  (gently curved down)
Scale:  100%
Glow:   40% opacity
Tilt:   0°
Use:    Default resting state
```

#### Happy / Pleased

```
Lid:    ⌣ ⌣  (upturned, "smiling")
Scale:  110%
Glow:   60% opacity
Tilt:   0°
Use:    Positive moments, celebrations
```

#### Curious / Interested

```
Lid:    ─ ─  (wide open, flat)
Scale:  115%
Glow:   50% opacity
Tilt:   3° right (head tilt)
Use:    Questions, new information
```

#### Concerned / Empathetic

```
Lid:    ∩ ∩  (slightly furrowed)
Scale:  95%
Glow:   45% opacity
Tilt:   -2° forward lean
Use:    User distress, sensitive topics
```

#### Thinking / Processing

```
Lid:    ⌒ ⌒  (neutral)
Scale:  100%
Glow:   35% opacity (dim)
Tilt:   5° left, upward
Use:    Processing, considering
```

#### Excited / Celebrating

```
Lid:    ◠ ◠  (wide, lifted)
Scale:  120%
Glow:   80% opacity
Tilt:   Bouncing (±2°)
Use:    Big wins, milestones
```

#### Listening / Attentive

```
Lid:    ⌒ ⌒  (relaxed but focused)
Scale:  105%
Glow:   50% opacity
Tilt:   -3° forward lean
Use:    During user speech
```

#### Sleepy / Night Mode

```
Lid:    ⌢ ⌢  (heavy, drooping)
Scale:  90%
Glow:   25% opacity
Tilt:   0°
Use:    Late night conversations
```

#### Surprised

```
Lid:    ○ ○  (wide open, circular)
Scale:  125%
Glow:   70% opacity
Tilt:   2° back
Use:    Unexpected news
```

#### Warm / Affectionate

```
Lid:    ⌣ ⌣  (soft smile)
Scale:  105%
Glow:   55% opacity, warm color shift
Tilt:   0°
Use:    Connection moments, callbacks
```

### Expression Intensity Levels

Each expression has 3 intensity levels:

| Level | Scale Multiplier | Glow Multiplier | Duration |
|-------|------------------|-----------------|----------|
| Subtle | 0.5x | 0.7x | 100ms |
| Normal | 1.0x | 1.0x | 250ms |
| Exaggerated | 1.5x | 1.3x | 400ms |

## 5. Animation Guidelines

### The Breathing System

Ferni breathes. Always. This is the foundation of feeling alive.

```javascript
// Breathing parameters
const BREATH_CYCLE = {
  idle: {
    duration: 4000,      // ms per breath
    scaleY: [1, 1.018],  // 1.8% vertical expansion
    scaleX: [1, 0.994],  // Inverse horizontal (volume constant)
    easing: 'gentle'
  },
  excited: {
    duration: 2500,
    scaleY: [1, 1.025],
    scaleX: [1, 0.985],
    easing: 'spring'
  }
}
```

### Squash & Stretch

Volume remains constant during deformation:

```
Normal:     Squashed:      Stretched:
   ○           ⬭              ◯
   ↑           ↑              ↑
  1:1        1.2:0.8        0.8:1.2
```

### Anticipation

Every action has a wind-up:

```
Action: Jump
  1. Anticipation: Scale down 5%, 80ms
  2. Action: Jump up 15px, 200ms spring
  3. Settle: Overshoot 3%, settle 150ms
```
