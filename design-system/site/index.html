Quick Start

Get started in minutes.

Install the design system and start building brand-compliant interfaces.

# Install npm install @ferni/design-system # Import tokens import '@ferni/design-system/tokens.css'; # Import components import { Avatar, toast, celebration } from '@ferni/design-system/components'; # Use brand-compliant messaging import { getRandomPhrase } from '@ferni/design-system/content-utils'; toast.success(getRandomPhrase('celebrations', 'smallWin')); // "You did it." or "Nice." etc.
