How It Actually Works:

  • Eyes: White gradient ellipse shapes (rx="8" ry="10") - NO pupils!
  • Sparkles: Tiny 2px white circle on each eye
  • Expression: CSS transforms on .eye-main (scaleY, scaleX, translate, rotate)
  • Window Lids: SVG path filled with var(--paper-cream)
  • Speaking: Both top AND bottom lids animate + body squash/stretch
