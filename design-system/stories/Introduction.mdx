# 🌿 Ferni Design System

Welcome to the Ferni Design System Storybook. This is the living documentation for all UI components, design tokens, and patterns used across Ferni products.

## Philosophy

Ferni's design system embodies:
- **Warmth** - Earthy, organic colors that feel approachable
- **Clarity** - Clean typography and generous whitespace (Ma)
- **Humanity** - Rounded corners, gentle animations, natural language

## Theme Support

Toggle between light and dark themes using the toolbar above. Each persona has its own color palette that adapts to the active theme.

## Accessibility

All components are designed to meet WCAG 2.1 AA standards:
- Minimum 4.5:1 contrast ratio for text
- Focus indicators on all interactive elements
- Keyboard navigation support
- Screen reader compatibility
