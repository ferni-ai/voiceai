# Relationship Context Builders

> **Building genuine connection over time**

This folder contains builders that manage relationship development, trust, and social awareness.

## Structure

```
relationship/
├── arc/                    # Relationship stage progression
│   ├── first-meeting-magic.ts      # Stranger (turns 0-3)
│   ├── acquaintance-deepening.ts   # Acquaintance (sessions 2-5)
│   ├── friendship-flowering.ts     # Friend (sessions 6-15)
│   └── trusted-advisor.ts          # Trusted Advisor (15+)
├── trust-context.ts        # Trust signals and boundary awareness
├── deep-relationship.ts    # Shared vocabulary, inside jokes, milestones
├── relationship-behaviors.ts # Relationship-appropriate behaviors
├── social-relationships.ts  # Social graph and connections
└── social-graph-context.ts  # Network visualization context
```

## Relationship Arc System

The arc system tracks relationship progression through stages:

1. **Stranger** - Energy matching, vulnerability modeling, noticing
2. **Acquaintance** - Callbacks, trust building, pattern recognition
3. **Friend** - Inside jokes, growth reflection, shared experiences
4. **Trusted Advisor** - Life arc awareness, values accountability

Each stage unlocks different behaviors and deeper connection patterns.
