# Engagement Context Builders

> **User engagement, games, music, and rituals**

This folder contains builders that make conversations engaging and fun - games, music integration, daily rituals, and storytelling.

## Builders

| Builder | Purpose |
|---------|---------|
| `engagement.ts` | General engagement patterns |
| `engagement-context.ts` | Current engagement state |
| `game-context.ts` | Games and playful interactions |
| `music.ts` | Music playback integration |
| `music-emotion-offers.ts` | Emotion-based music suggestions |
| `daily-rituals.ts` | Morning Sky Check, Habit Heartbeat, etc. |
| `storytelling.ts` | Narrative and storytelling patterns |
| `referral-prompt.ts` | Natural referral opportunities |

## Daily Rituals

The daily-rituals builder enables features like:
- **Morning Sky Check** - Weather-aware morning greeting
- **Habit Heartbeat** - Gentle habit check-ins
- **Evening Reflection** - End-of-day processing
