# Learning Context Builders

> **Collective intelligence and wisdom synthesis**

This folder contains builders that enable learning from the community (anonymized) and synthesizing wisdom.

## Builders

| Builder | Purpose |
|---------|---------|
| `community-learning.ts` | Anonymous patterns that help everyone |
| `wisdom-synthesis.ts` | Synthesizing wisdom from conversations |

## Philosophy

Learning should be:
- **Anonymous** - No personal data crosses users
- **Helpful** - Patterns that genuinely help
- **Opt-in** - User consent for contributions

## What We Learn

- What coaching approaches work for similar situations
- Common patterns in growth and change
- Effective ways to frame difficult topics

- `intelligence/` - Pattern detection feeds learning
