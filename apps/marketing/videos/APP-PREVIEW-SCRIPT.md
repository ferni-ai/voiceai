# 🎬 Voice AI - App Preview Video Scripts

Scripts for generating App Store preview videos using Veo 3 and screen recordings.

---

## 📱 iOS App Store Preview (15-30 seconds)

### Technical Requirements
- **Duration**: 15-30 seconds
- **Resolution**: 1920x1080 (landscape) or 1080x1920 (portrait)
- **Frame Rate**: 30fps
- **Audio**: Optional but recommended

## 🎬 Production Notes

### Screen Recording Tips

1. **Clean device**: Hide personal info, notifications off
2. **Slow down**: Record at 0.5x speed, speed up in edit
3. **Tap indicators**: Enable touch visualization
4. **Consistent lighting**: Match Veo footage warmth

### Veo Generation Tips

1. **Batch generate**: Run all prompts, pick best
2. **Match color palette**: Always include hex codes
3. **Request loops**: Easier for seamless backgrounds
4. **4K minimum**: Downscale as needed

### Audio Recommendations

- **Background**: Soft ambient/lo-fi (royalty-free)
- **No voiceover**: Let text tell the story
- **Sound effects**: Subtle UI sounds optional

## 📁 Export Settings

### iOS App Store
- 1920x1080 or 1080x1920
- H.264, 30fps
- Max 500MB
- No App Store badges in video

### Google Play
- 1920x1080 landscape
- YouTube link (upload separately)
- 30-120 seconds

### Social
- 1080x1920 (vertical)
- Under 60 seconds
- Auto-captions recommended
