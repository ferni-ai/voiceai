## Animation Inventory

### Available Animation Tokens (from design system)

#### Durations
| Token | Value | Use Case |
|-------|-------|----------|
| `micro` | 50ms | Immediate feedback |
| `fast` | 100ms | Hover states |
| `normal` | 200ms | Standard transitions |
| `slow` | 300ms | Deliberate moves |
| `moderate` | 400ms | Panel slides |
| `deliberate` | 500ms | Emphasis |
| `dramatic` | 600ms | Celebrations |
| `celebration` | 800ms | Major events |
| `glacial` | 1500ms | Ambient effects |

#### Easings
| Token | Description |
|-------|-------------|
| `spring` | Pixar bounce |
| `springGentle` | Subtle bounce |
| `expoOut` | Dramatic exit |
| `gentle` | Organic, natural |
| `anticipate` | Wind-up |
| `elastic` | Snap effect |

#### Persona Animation Profiles
| Persona | Speed | Easing | Feel |
|---------|-------|--------|------|
| Ferni | 1.0x | gentle | Warm, balanced |
| Jack | 1.2x | gentle | Deliberate, wise |
| Peter | 0.85x | spring | Quick, energetic |
| Alex | 1.0x | standard | Professional |
| Maya | 0.95x | standard | Efficient |
| Jordan | 0.8x | spring | Celebratory |
