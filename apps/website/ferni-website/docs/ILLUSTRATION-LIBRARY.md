## Usage Notes

### Generating Images
1. Copy the prompt text (after the filename)
2. Use Google Imagen 3 via Vertex AI or the generate-assets.js script
3. Set aspect ratio to 16:9 for blog, 1:1 for social
4. Save with the suggested filename

### Customizing Prompts
- Swap colors while staying in palette
- Add specific elements relevant to content
- Adjust mood words to match article tone
- Keep the base aesthetic references

### Batch Generation
All prompts in this library can be added to `scripts/generate-assets.js` for batch generation.
