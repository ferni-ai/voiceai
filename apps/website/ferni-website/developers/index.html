Why Ferni Agents?

A complete system for building conversational AI agents

Bundle System

Self-contained agent packages with identity, voice, behaviors, and knowledge. Drop in a folder, restart, done.

CLI Tools

Create agents from templates, validate bundles, and manage your team roster from the command line.

Voice Integration

Seamless Cartesia voice synthesis with per-agent voice IDs, speech characteristics, and emotional expression.

Smart Handoffs

Agents automatically hand off to each other based on triggers. Define keywords, the system does the rest.

Lazy Loading

Content loaded on demand. Stories, knowledge, and behaviors only fetched when needed.

Full TypeScript

100% TypeScript with strict typing, excellent IDE support, and runtime validation with Zod.

Quick Start

Add a new agent to your team in under 5 minutes

1

Clone the repository

Get the Ferni Agents framework

git clone https://github.com/ferni-ai/ferni-agents.git
cd ferni-agents
npm install
2

Create your first agent

Use the CLI to scaffold a new agent from a template

# Create a sage-style advisor
npm run agents create my-advisor --template sage

# Or a specialist for a specific domain
npm run agents create tax-expert --template specialist
3

Start talking

Run the development server and connect

# Start with your new agent
PERSONA_ID=my-advisor npm run dev

# Open the client
open http://localhost:3000
