## On-Page SEO Checklist

For every article:

### Title & Meta
- [ ] Primary keyword in title (first 60 chars)
- [ ] Meta description includes keyword (155 chars)
- [ ] URL slug is clean and keyword-rich
- [ ] Title is compelling (not just keyword-stuffed)

### Headers
- [ ] H1 matches page title (one per page)
- [ ] H2s include related keywords
- [ ] H3s break up long sections
- [ ] Table of contents for articles >2,000 words

### Content
- [ ] Primary keyword in first 100 words
- [ ] Keyword density 1-2% (natural, not forced)
- [ ] Related keywords throughout
- [ ] Internal links to 3-5 other blog posts
- [ ] External links to 2-3 authoritative sources
- [ ] Images with alt text including keywords

### Technical
- [ ] Page loads in <3 seconds
- [ ] Mobile-responsive layout
- [ ] Schema markup (Article, FAQ where relevant)
- [ ] Canonical URL set
- [ ] No duplicate content issues

### Engagement
- [ ] Clear CTA to try Ferni (at least 2 per article)
- [ ] Related posts section at bottom
- [ ] Social share buttons
- [ ] Email capture opportunity

## Link Building Strategy

### Quick Wins (Month 1-2)

1. **Directory Submissions**
   - Product Hunt (launch + daily upvote collection)
   - AlternativeTo (create Ferni page)
   - G2 Crowd
   - Capterra
   - GetApp
   - SaaSHub

2. **Profile Links**
   - Crunchbase company profile
   - LinkedIn company page
   - Twitter/X with link
   - YouTube About page (if creating video content)
   - Medium publication

3. **Founder Profiles**
   - Personal LinkedIn with company mention
   - Twitter/X bio
   - Any speaking/podcast bios

### Earned Links (Month 2-4)

1. **HARO/Qwoted/SourceBottle**
   - Monitor for queries about: AI, coaching, mental health tech, productivity apps
   - Respond within 2 hours for best results
   - Aim for 5-10 pitches per week
   - Target: 2-3 links per month

2. **Guest Posts**
   - Target sites: Medium publications, productivity blogs, mental health blogs
   - Topics: "How AI is Changing Personal Development", "The Future of Coaching"
   - Include link to relevant Ferni blog post (not homepage)
   - Target: 2 guest posts per month

3. **Podcast Appearances**
   - Pitch to: self-improvement, productivity, tech, mental health podcasts
   - Topics: founder story, AI coaching space, personal transformation
   - Target: 1-2 podcasts per month

### Relationship Building (Ongoing)

1. **Blogger Outreach**
   - Find bloggers who've written about Replika, Pi, AI companions
   - Offer product access for honest review
   - Don't ask for links directly - let quality drive mentions

2. **Resource Page Outreach**
   - Find pages like "Best AI Tools for X"
   - Pitch for inclusion with specific value prop
   - Example: "50 AI Tools for Productivity" → pitch Ferni as coaching angle

```
└── /contact/ (contact page)
```

### Required Pages
- [ ] XML Sitemap (auto-generated)
- [ ] robots.txt (allow all, sitemap reference)
- [ ] 404 page with search/navigation
- [ ] Privacy policy
- [ ] Terms of service

### Schema Markup

**Organization Schema** (homepage):
```json
{
  "@context": "https://schema.org",
  "@type": "Organization",
  "name": "Ferni",
  "url": "https://ferni.ai",
  "logo": "https://ferni.ai/logo.png",
  "description": "AI life coaching platform with infinite memory",
  "sameAs": [
    "https://twitter.com/tryferni",
    "https://linkedin.com/company/ferni"
  ]
}
```

**Article Schema** (each blog post):
```json
{
  "@context": "https://schema.org",
  "@type": "Article",
  "headline": "Article Title",
  "author": {
    "@type": "Organization",
    "name": "Ferni"
  },
  "datePublished": "2026-01-15",
  "dateModified": "2026-01-15"
}
```

**FAQ Schema** (where relevant):
```json
{
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "mainEntity": [{
    "@type": "Question",
    "name": "What is an AI life coach?",
    "acceptedAnswer": {
      "@type": "Answer",
      "text": "An AI life coach is..."
    }
  }]
}
```
