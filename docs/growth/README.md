## Priority Order (Recommended)

### Phase 1: Foundation (Weeks 1-4)

1. **TikTok** - Start immediately, compounds over time
2. **Reddit** - Build karma while other channels spin up
3. **SEO** - Publish first 4 articles

### Phase 2: Launch (Weeks 5-8)

4. **Product Hunt** - Execute launch
5. **Influencer** - Begin nano-tier outreach
6. **Continue** - TikTok, Reddit, SEO

### Phase 3: Scale (Weeks 9-12)

- Double down on winning channels
- Add paid amplification to organic winners
- Begin micro/mid-tier influencer partnerships

## Weekly Rhythm

### Monday: Strategy
- Review last week's metrics
- Plan content for the week
- Approve influencer content

### Tuesday-Thursday: Execution
- TikTok: Film/edit 5+ videos
- SEO: Write 1-2 articles
- Reddit: Value posts + comments
- Influencer: Outreach batch

### Friday: Outreach
- Influencer follow-ups
- Community engagement
- Schedule weekend content

### Weekend: Light Touch
- Engage with comments
- Monitor/respond
- Rest and recharge
