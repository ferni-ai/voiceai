# 🌱 Ferni Coming Soon Roadmap Experience

## Transform Incomplete Features into Inspiring Promises

**Version 1.0 | December 2024**

---

## Executive Summary

Instead of showing incomplete features as "locked" or hidden, we transform them into a **beautiful roadmap experience** that builds excitement and aligns with Ferni's brand promise: _"Better than human."_

This plan transforms menu items that aren't ready into a cohesive "What's Growing" experience that:

1. Builds anticipation and emotional investment
2. Communicates Ferni's vision and values
3. Lets users influence priority through voting
4. Creates a sense of shared journey

## 🌿 The "What's Growing" Experience

### Brand Philosophy

> _"Every great relationship has seasons. These are seeds we're planting together."_

Instead of "Coming Soon" (tech speak), we use:

- **"Growing"** - Features in active development
- **"Planted"** - Features planned but not started
- **"Dreaming"** - Features we're exploring
- **"Blooming Soon"** - About to launch

### Visual Language

| Element     | Design Token | Description    |
| ----------- | ------------ | -------------- |
| **Seeds**   | 🌱           | Earliest stage |
| **Sprouts** | 🌿           | In development |
| **Buds**    | 🌸           | Almost ready   |
| **Bloom**   | 🌺           | Launching soon |

## 🎨 UI Implementation

### New Component: `RoadmapPanel`

Replace locked feature clicks with a beautiful roadmap modal:

```typescript
// apps/web/src/ui/roadmap-panel.ui.ts

interface RoadmapFeature {
  id: string;
  headline: string;
  description: string;
  stage: 'seed' | 'sprout' | 'bud' | 'bloom';
  superhuman: string[];
  existing?: string[];
  estimatedArrival: string;
  canVote: boolean;
  voteCount?: number;
}
```

### Menu Integration

When a roadmap feature is clicked:

1. Don't shake as "locked"
2. Open the Roadmap Panel with that feature highlighted
3. Show the brand-aligned description
4. Allow voting/interest registration
5. Show estimated timeline

### Visual Design

```css
/* Roadmap card stages */
.roadmap-card--seed {
  --stage-color: var(--color-muted);
  --stage-icon: '🌱';
}
.roadmap-card--sprout {
  --stage-color: var(--persona-primary);
  --stage-icon: '🌿';
}
.roadmap-card--bud {
  --stage-color: var(--color-warning);
  --stage-icon: '🌸';
}
.roadmap-card--bloom {
  --stage-color: var(--color-success);
  --stage-icon: '🌺';
}
```
