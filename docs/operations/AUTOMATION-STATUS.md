## Runbook: Common Issues

### Agent Not Joining Rooms

1. Check logs: `npm run ops:logs:errors`
2. Look for "runner initialization timed out"
3. Fix: Ensure `numIdleProcesses: 1` in `voice-agent.ts`

### High Error Rate

1. Check error rate: `npm run ops:logs:errors`
2. Check if specific persona failing
3. Common cause: Missing `content` section in persona manifest

### Service Down

1. Run health check: `npm run ops:health`
2. If down, check Cloud Run console
3. Manual restart: Redeploy via `npm run deploy:agent:async`

### Cost Spike

1. Check GCP Billing console
2. Look for unexpected instance scaling
3. Check for infinite loops in agent code
