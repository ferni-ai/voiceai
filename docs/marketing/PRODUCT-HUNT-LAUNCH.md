## Answers to Common Questions

### Q: How is this different from ChatGPT?
```
Great question! ChatGPT is text-based. Ferni agents have real-time voice conversations - you literally talk to them like a phone call. Sub-200ms latency, natural interruptions, the whole thing. Plus, you fully customize the personality, expertise, and deploy on your own domain.
```

### Q: What about privacy/security?
```
Conversations are encrypted in transit. We're working on SOC 2 certification. Enterprise plans offer on-prem deployment. You can also self-host the entire stack.
```

### Q: Can I use my own voice?
```
Yes! Through Cartesia's voice cloning. Upload 30 seconds of audio and create a custom voice. Great for legacy preservation use cases.
```

### Q: What's the latency like?
```
Sub-200ms end-to-end, which is faster than human reaction time. We use LiveKit for WebRTC, so it feels like a real phone call.
```

### Q: Do I need ML experience?
```
Nope! If you can write Markdown and JSON, you can build an agent. The wizard handles everything else.
```

### Q: What LLM does it use?
```
OpenAI by default, but you can configure Gemini, Anthropic, or others. The personality and expertise come from your system prompt, not the base model.
```

### Q: How much does it cost to run an agent?
```
Free tier: 100 minutes/month (great for testing)
Pro: $49/month for 1,000 minutes
At scale: roughly $0.05-0.10 per minute

Cloud Run only charges when agents are in use, so idle agents cost ~$0.
```
