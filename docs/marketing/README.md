## Recording Guide

### Tools
- **Terminal recording:** asciinema + svg-term-cli
- **Screen recording:** OBS or ScreenFlow
- **GIF conversion:** gifski or Ezgif
- **Editing:** DaVinci Resolve (free)

### Settings
- Terminal: 80x24, 18-20px font, Dracula theme
- Screen: 1920x1080, hide personal info
- Audio: -16 LUFS, quiet room

### Tips
- Type deliberately (not too fast)
- Pause after commands complete
- Show real output, not mocked

## Platform-Specific Notes

### Twitter
- Threads with GIFs perform best
- 9 AM PT optimal posting time
- Quote RT cool user creations
- Avoid hashtag spam (1-2 max)

### LinkedIn
- Professional tone, but still friendly
- Longer posts (1,300+ chars) get more reach
- Images > text-only
- Tag relevant people sparingly

### Product Hunt
- Launch at 12:01 AM PT
- Respond to every comment
- Don't ask for upvotes explicitly
- Hunter helps but not required

### Hacker News
- Technical, honest tone
- Acknowledge limitations
- Don't be defensive
- Never ask for upvotes

### Dev.to
- Code blocks are key
- Tutorial format works best
- Series can extend reach
- Engage in comments
