## API Endpoint Summary

### Video Sessions (`/api/video/*`)

| Endpoint                        | Method | Description            |
| ------------------------------- | ------ | ---------------------- |
| `/api/video/capabilities`       | GET    | Get supported features |
| `/api/video/state`              | GET    | Current session state  |
| `/api/video/enable`             | POST   | Enable camera          |
| `/api/video/disable`            | POST   | Disable camera         |
| `/api/video/screen-share/start` | POST   | Start screen sharing   |
| `/api/video/screen-share/stop`  | POST   | Stop screen sharing    |
| `/api/video/mode`               | POST   | Set display mode       |
| `/api/video/config`             | POST   | Update settings        |

### Wearable Integration (`/api/wearable/*`)

| Endpoint                         | Method | Description        |
| -------------------------------- | ------ | ------------------ |
| `/api/wearable/status`           | GET    | Connection status  |
| `/api/wearable/connect`          | POST   | Initiate OAuth     |
| `/api/wearable/disconnect`       | POST   | Remove provider    |
| `/api/wearable/data`             | GET    | Aggregated metrics |
| `/api/wearable/stress`           | GET    | Stress indicators  |
| `/api/wearable/sleep`            | GET    | Sleep analysis     |
| `/api/wearable/activity`         | GET    | Activity summary   |
| `/api/wearable/coaching-context` | GET    | Context for AI     |
| `/api/wearable/sync`             | POST   | Trigger sync       |
| `/api/wearable/config`           | POST   | Update settings    |

### Group Coaching (`/api/group/*`)

| Endpoint                         | Method | Description     |
| -------------------------------- | ------ | --------------- |
| `/api/group/sessions`            | GET    | List sessions   |
| `/api/group/sessions`            | POST   | Create session  |
| `/api/group/sessions/:id`        | GET    | Session details |
| `/api/group/sessions/:id/start`  | POST   | Start session   |
| `/api/group/sessions/:id/end`    | POST   | End session     |
| `/api/group/sessions/:id/join`   | POST   | Join session    |
| `/api/group/sessions/:id/topics` | POST   | Add topic       |
| `/api/group/sessions/:id/goals`  | POST   | Add goal        |
