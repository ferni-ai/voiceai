# 🔧 QUICK GENERATION WORKFLOW

## Option 1: Google AI Studio (Recommended)
1. Go to https://aistudio.google.com
2. Select Imagen 3 for images
3. Copy prompts from above
4. Download generated assets
5. Move to appropriate folders

## Option 2: Midjourney
1. Join Discord: https://discord.gg/midjourney
2. Use /imagine command with prompts above
3. Upscale favorites (U1-U4)
4. Download and organize

## Option 3: Node.js Script
```bash
cd promo/ferni-website
npm install @google-cloud/aiplatform
export GOOGLE_APPLICATION_CREDENTIALS="/path/to/key.json"
node scripts/generate-assets.js --batch=images
```
