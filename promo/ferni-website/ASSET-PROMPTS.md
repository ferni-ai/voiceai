## 📐 TECHNICAL SPECIFICATIONS

### Video Specs (Hero Background)
- **Resolution:** 4K (3840x2160) minimum, 8K preferred for future-proofing
- **Frame Rate:** 24fps for cinematic feel, 30fps acceptable
- **Duration:** 10-15 seconds
- **Loop:** Seamless preferred
- **Format:** MP4 (H.264/H.265) for web, ProRes for master
- **File Size Target:** Under 10MB for web delivery

### Image Specs
- **Format:** WebP with PNG fallback
- **DPI:** 72 for web, 144 for retina
- **Compression:** Optimized for web (TinyPNG)

## 💡 VEO 3 TIPS

1. **Always specify:**
   - Resolution: "8K", "4K"
   - Style: "cinematic", "Apple-quality", "premium"
   - Speed: "slow motion", "timelapse"
   - Camera: "drone shot", "macro", "static", "slow push"

2. **Color consistency:**
   - Reference exact hex codes
   - Mention "warm cream", "forest green", "amber" 

3. **Mood keywords:**
   - "contemplative", "peaceful", "premium"
   - "minimal", "elegant", "sophisticated"

4. **Avoid:**
   - "stock photo", "generic"
   - Busy compositions
   - Cool blue tones (not our brand)
