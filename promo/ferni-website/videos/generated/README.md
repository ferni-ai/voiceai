# Generated Videos

This folder contains AI-generated videos for the Ferni landing page.

## Folder Structure

```
generated/
├── hero/              # Hero section background videos
├── testimonials/      # Customer testimonial videos
└── backgrounds/       # Ambient background loops
```

## Generation Methods

### Option 1: Google Veo 2 (via AI Studio)
1. Go to https://aistudio.google.com
2. Select Veo 2 model (if available)
3. Copy prompts from /prompts/PLATFORM-SPECIFIC-PROMPTS.md

### Option 2: Runway Gen-3
1. Go to https://runwayml.com
2. Use Gen-3 Alpha model

### Option 3: Pika Labs
1. Use web app or Discord

## Video Requirements

| Type | Duration | Resolution | Format |
|------|----------|------------|--------|
| Hero Loop | 10-15s | 4K/1080p | MP4 |
| Testimonial | 15-30s | 1080p | MP4 |
| Background | 10-30s | 1080p | MP4 |

## Color Palette

All videos should use warm earth tones:
- Cream (#F5F1E8)
- Sage Green (#4a6741)
- Golden Amber (#C4A265)
- Cedar Brown (#9a7b5a)
